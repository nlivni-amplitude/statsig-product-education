How skills consume this

Skills published at amplitude/learning-experience-internal reference these files at query time via WebFetch to URLs like:

https://amplitude.github.io/statsig-product-education/topics/05-experiments.md
https://amplitude.github.io/statsig-product-education/indexes/feature-comparison-inventory.json

This means:

  • Skills are small (one .md file each) and distributable without a plugin install
  • Content updates here propagate immediately on the next fetch — no re-install needed
