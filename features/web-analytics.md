# Web Analytics

**What it is:** Lightweight web autocapture surface: page views, clicks, form submits without manual instrumentation. Lower setup cost than full event-design, lower fidelity.

## Amplitude counterpart

- **Concept:** Amplitude Analytics (autocapture)
- **Match quality:** partial
- **Key differences:**
  - Both offer autocapture as a faster path to first events
  - Statsig's web analytics is newer; Amplitude's autocapture is more mature
